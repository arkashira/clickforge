# breakeven.md

## Unit Economics & Break-even Analysis for Clickforge

### Cost per Active User
- **Compute Costs**: $0.15/user/month
- **Storage Costs**: $0.05/user/month
- **Bandwidth Costs**: $0.10/user/month
- **Total Cost per Active User**: 
  - **Total**: $0.15 + $0.05 + $0.10 = **$0.30/user/month**

### Pricing Tiers
| Tier          | Price ($/mo) | Features                                                                 |
|---------------|---------------|--------------------------------------------------------------------------|
| **Basic**     | $10           | Basic browser automation, limited integrations, email support           |
| **Pro**       | $25           | Advanced automation features, integrations with DevSecOps tools, chat support |
| **Enterprise** | $50           | Custom solutions, priority support, dedicated account manager, advanced analytics |

### Customer Acquisition Cost (CAC) Range
- **CAC Estimate**: $50 - $100 per user
  - This includes marketing expenses, sales efforts, and onboarding costs.

### Lifetime Value (LTV) Estimate
- **Average Revenue per User (ARPU)**: 
  - Basic: $10
  - Pro: $25
  - Enterprise: $50
- **Average Customer Lifespan**: 24 months
- **LTV Calculation**:
  - Basic: $10 * 24 = **$240**
  - Pro: $25 * 24 = **$600**
  - Enterprise: $50 * 24 = **$1200**

### Break-even Users Count
- **Monthly Fixed Costs**: $3,000 (including salaries, marketing, and operational costs)
- **Contribution Margin per User**:
  - Basic: $10 - $0.30 = **$9.70**
  - Pro: $25 - $0.30 = **$24.70**
  - Enterprise: $50 - $0.30 = **$49.70**
  
- **Break-even Users Count**:
  - Basic: $3,000 / $9.70 ≈ **310 users**
  - Pro: $3,000 / $24.70 ≈ **122 users**
  - Enterprise: $3,000 / $49.70 ≈ **60 users**

### Path to $10K MRR
- **Target Monthly Recurring Revenue (MRR)**: $10,000
- **Required Users for Each Tier**:
  - Basic: $10,000 / $10 = **1,000 users**
  - Pro: $10,000 / $25 = **400 users**
  - Enterprise: $10,000 / $50 = **200 users**

### Summary
- **Most Viable Tier for Growth**: Pro Tier with 400 users, balancing affordability and feature set.
- **Break-even Analysis**: Focus on acquiring users in the Pro Tier to optimize profitability while minimizing CAC.
